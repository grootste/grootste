## 📌 Featured Projects

### 🫀 [Intelligent Healthcare Monitoring System with Pulse Oximetry Analysis](https://github.com/grootste/Intelligent-Healthcare-Monitoring)
> **Final Year Project · VIT 2021 · Python · Arduino · Machine Learning · IoT**

Built an end-to-end IoT health monitoring system using **Arduino Uno** for real-time pulse oximetry acquisition (SpO₂ + heart rate via MAX30100 sensor), paired with a **supervised ML classification pipeline** in Python to stratify patients across multiple health risk levels.

- 🔬 Cross-disciplinary integration: embedded hardware → signal processing → ML inference
- 🧠 Trained & evaluated Random Forest, SVM, KNN, and Decision Tree classifiers (~92% accuracy)
- 📡 Serial communication bridge between C++ firmware and Python inference engine
- ⚠️ 4-level risk stratification (Normal / Low / Moderate / High) with clinically inspired thresholds

---

### 🦯 [Smart Stick for Visually Impaired](https://github.com/grootste/Smart-stick)
> **Academic Project · VIT 2020 · Arduino · C++ · IoT · GPS · GSM (SIM808)**

Designed a sensor-based assistive navigation device using **ultrasonic and water sensors** to detect obstacles, puddles, and hazards in real time. Integrated a **SIM808 GPS/GSM module** to send live location via Google Maps SMS to a caregiver, with a physical SOS button triggering emergency alerts.

- 🛰️ Real-time GPS tracking transmitted via GSM SMS
- 🚧 Multi-hazard detection: obstacles, water, uneven terrain
- 🆘 One-touch SOS emergency alert system

### 🎭 [Playwright Automation Framework](https://github.com/grootste/playwright-framework)
> **Professional Project · · Playwright · TypeScript · AI Agents · CI/CD**

Production-grade Playwright test automation framework built from scratch at Propertystack, incorporating **AI Agents for test planning, script generation, and self-healing locators**. Includes 100+ reusable page functions, parallel execution, and a 5-step locator strategy.

- 🤖 AI-assisted test generation and self-healing locator fallbacks
- ⚡ Parallel test execution across browser contexts
